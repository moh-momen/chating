Server should be run first always and wait for connections form clients
Many clients can access Clients to server

For server run this commands:
javac Server.java
java Server

For Client run this commands:
Client1:
javac Client.java
java Server

Client2:
javac Client.java
java Server


Client3:
javac Client.java
java Server

Client4:
javac Client.java
java Server

Client5:
javac Client.java
java Server

and so on.
